# 🍮 푸딩커비톡
푸딩 벚꽃톡(가제)은 메세지 채팅을 기본으로 하는 웹 서비스로
채팅 이외에도 챗봇, 할 일 목록, 재미있는 1일 1회 추첨 이벤트, 포토 앨범(예정) 등의
다양한 라이프 서비스를 제공하는 복합형 웹 서비스 입니다.<br/> <br/> <br/>
